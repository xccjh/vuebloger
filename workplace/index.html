面试 2021-01-30
体制内职场学问

编者按 : 体制内发展，靠的是背景、资源、部门、性格、年龄、性别、机遇……加努力，努力只是其中的一个方面。

但，如果你没有投个好胎，你唯一能把握的东西，就是努力了。 本文梳理的十条特质，很有说服力，值得细品。

# 一、沉稳但不木讷

这是当官的最基本素质，也就是说，做人做事得稳重、靠谱，值得信赖。这里面有几个要素。一是低调做人，不能啥事没办就亚麻跌、噶的耶，没完没了不敬业。二是闷头做事，有板有眼、有始有终、有执行力，从不PE，更不ED。三是依章依规，科学有素、得当有度，不会用裤裆藏雷，更不会傻到用手榴弹炸飞机。

我曾经一个老领导，四十出头怼到副厅级，就属于沉稳这一类人。此君平时不显山不漏水，领导安排的啥事从来都是应声接下，然后一边干一边找依据，一个阶段一个汇报，就像领导的两只腿和一双眼，他干得清清楚楚、磊落光明，领导也跟着明明白白、放心安心。去年底被放到一个省辖市当了副市长。

但沉稳不代表木讷、不灵光，不代表死板、不创新。整天就那一副脸，把自己装在套子里，谁看了都会生厌；啥事都是按图索骥，不敢越雷池一步，谁都不会赋以大任。这样的人只适合按部就班，不适合火线提拔。

职场 2019-07-07
面试时问题一览

# 技术面1

# 一、异步编程的常用方法有哪些？

  1. Promise
  2. Async await
  3. Generator函数
  4. rxjs
  5. 消息发布订阅机制
  6. 事件监听
  7. 回调函数

# 二、Argument的callee，Function的Caller区别

  1. Caller是函数调用者
  2. Callee是调用函数本身

# 三、js数据类型有哪些，Es6新增的数据类型有哪些？

数据类型：null,string,undefine,boolean,object,number 新增数据类型：symbol（解决属性名相同的问题）set,map等

# 四、vue进行组传值的方式有哪些？

  1. $emit event和pass props方式
  2. 借助另外一个vue实例的event-bus方式
  3. 使用第三方库 pubsub-js ，PubSub –subscribe订阅和PubSub publish发布传参
  4. 使用slot插槽
  5. 使用vuex

# 五、你觉得Vue的编程哲学是什么？

  1. 数据驱动，面向对象的组件化开发，融合了angular双向数据绑定，结构与逻辑分离的指令系统，html补丁的编程思想，标签传值等和react组件化，虚拟DOM，模仿redux单向数据流的状态管理器，是个声明式，渐进式MVVM框架。（自己随便组织）
