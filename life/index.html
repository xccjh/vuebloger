# 前端修炼之路

一步，两步，三步四步五步，到达人生的一个开发境界~

# 初级前端

掌握以下知识点：

  1. 结构/样式/行为/数据：html/ps/css/javaScrpt/（ajax/fetch）,mockjs/easymock,amock,json-server，postman

  2. js进阶设计模式/框架设计/数据结构，事件系统,原型/es5/6/7/8/9/10/next，aysnc/await/genarator/pomise/set/map等等,innodb前端数据库，异步事件队列编程/v8引擎高并发方案，Webwork多线程编程，内存泄露，尾递归优化，gulp自建开发环境等等等

  3. 样式进阶前/后预理器scss/less/stylus的变量，函数，mixin,混入等等，常用代码片段，BFC/haslayout/csshack/媒体查询，flex/-wibket-box/grid/table布局等等

  4. 数据进阶socket主动推送，H5的Server-sent事件，长短连接/轮询,计划任务每整点执行等等

  5. 常用行为框架：jq，jQuery UI/mobile ,jq系列插件等Widget部件等等,zepto,lodash,momentjs

  6. 常用ui框架：bootstrap，easyui，layui，swiper，weui,公司eapui等等

  7. 大数据展示webgis：echarts，highchart,antv/g2，leafet，openlayer,百度/高德/腾讯地图以及天地图等等

  8. 移动端适配vp/rem/vh等等,响应式开发，浏览器兼容reset/normal/prefix/profill，浏览器技巧/调试等，网页性能优化，压缩，精灵图，

  9. dom1/2/3/4规范，ecma262(javaScript)/400（json）接口规范，html规范，css分级规范,MDN中BOM文档等

  10. git高手之路，svn使用指南

  11. IDE指南，vscode、webstorm,subline插件等等

# 前端标准规范-v1.0

总体原则：极简、极快、解耦

主要适用范围：vue 单页项目+

# 文件组织规范

# 文件结构

原则：文件少、层级浅、资源集中、相对独立互不影响

基本结构：

.
|_ node_modules
|_ dist
|_ src
    |_ assets // 公共资源
        |_ img // 全局图片
            |- favicon.png
        |- common.js // 全局公用 js
        |- common.less // 重置样式表 + 自定义全局样式
        |- mixin.less // 全局 less 变量
        |- template.html // 模板
    |_ components // 组件
        |_toast
            |- index.vue
            |- index.less
        |- index.js // 所有组件的集合
    |_ pages // 具体页面
        |_ index // 首页
            |- index.vue
            |- index.less
            |- good.png // 少量本页面下独自使用的图片
        |- img // 两个以上页面都用到的图片单独抽离防止
            |- copy.jpg
        |_ temp // 模板
            |- index.vue
            |- index.less
    |_ router // 路由
        |- index.js
    |_ vuex // vuex 插件
        |- index.js
    |- App.vue
    |- main.js
