          # 1. 通过SVG2geoJSON（gulp）转换

          1. 下载git clone https://github.com/yuhonyon/svg2geojson.git
          2. 安装cd geojson&&npm i
          3. 将 svg 文件直接放在 ** svg/ ** 目录下输入命令gulp,在**dist/**目录下将生产对应的目录,目录下有转换后的geojson文件以及行政中心的坐标文件和svg原文件

          温馨提示

          注意svg坐标要有空格，关闭vscode格式化相关,把文件放在svg目录，运行gulp转换即可,注意SVG2geoJSON在github上clone到本地后要npm i安装依赖

          # 2. 通过pip安装gdal

          1. 第一种在线安装
```
pip install GDAL==1.6.1
```
2. 第二种本地安装

pip安装的gdal的`GDAL-3.0.4-cp38-cp38-win_amd64.whl`(类似文件名)文件,下载完成之后放到Python安装目录下面的`\Scripts`文件下面点击上面的路径，输入cmd后运行`pip install GDAL-2.2.4-cp36-cp36m-win_amd64.whl`即可安装GDAL库，安装之后需要使用import导入对应的库文件`from osgeo import gdal;from osgeo import osr`，然后就可以使用GDAL库

          # 5. 直接通过gdal文件

          通过gdal文件使用GDAL库和org2org。关于[gdal和python安装](https://www.douban.com/note/260438027/)

1. `http://pypi.python.org/pypi/GDAL/1.6.1`下载GDAL的python包`GDAL-1.6.1.win32-py2.5.exe`。安装`GDAL-1.6.1.win32-py2.5.exe`文件，自动安装到 python的安装目录`$:\Python25\Lib\site-packages`。

2. 到GDAL官网`http://www.gdal.org/`的`http://download.osgeo.org/gdal/win32/1.6/`下载`gdalwin32exe160.zip`，解压缩到任意目录下，比如我解压到C盘根目录下，如`C:\gdalwin32-1.6`，打开`C:\gdalwin32-1.6\README_EXE.TXT`，里面有配置环境变量的说明。按照README_EXE.TXT文件的说明，在“我的电脑”->"高级  "->"环境变量"。将`D:\gdalwin32-1.6\bin`添加到"Path"变量中。并新增一个环境变量GDAL_DATA，变量值设置为`D:\gdalwin32-1.6\data`
<center>![gdalwin32exe160下载](/vuebloger/img/post/Snipaste_2020-04-27_11-53-27.png 'gdalwin32exe160下载')</center>
<center>下载gdal</center>
3. ok,可以了,在pythonwin中验证下
```
from osgeo import gdal
dataset = gdal.Open("g:/中国12--分经纬度原图/2095-2196.jpg")
print dataset.RasterXSize

//结果输出：3328
```
好了 开始研究GDAL库吧
