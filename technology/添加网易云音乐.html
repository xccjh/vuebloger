          # audio标签

          <!-- 添加背景音乐 不支持audio元素的浏览器会显示标签内文字-->
<audio id="music" src="https://music.163.com/#/song?id=3405868" autoplay preload="auto" loop="loop">
该浏览器不支持audio属性
</audio>

          属性 Value 描述
          src=“url” 音频地址
          autoplay=“autoplay” 如果出现该属性，则音频在就绪后马上播放
          controls=“controls” 如果出现该属性，则向用户显示控件，比如播放按钮
          loop=“loop” 如果出现该属性，则每当音频结束时重新开始播放
          preload=“preload” 出现该属性，则音频在页面加载时进行加载并预备播放。如果使用 “autoplay”，则忽略该属性

          # js函数控制

          // 自定义背景音乐的 播放/暂停 功能
function playMusic(){
    var music = document.getElementById("music");
    if (music.paused){
        music.play();//播放
    }else{
        music.pause();//暂停
        //music.load();//重新加载
    }
}

          # Chrome

          -Uncaught (in promise) DOMException: The element has no supported sources.

          # 解决方法一

          1. 点击播放–>查看Network–>找到以.mp3结尾的链接 !(找出音乐外链)['/vuebloger/img/post/20170613180036838.png' 找出音乐外链]
          2. 截取以.mp3结尾的链接 !(找出音乐外链)['/vuebloger/img/post/20170613180329171.png' 找出音乐外链]
          3. 嵌入到页面链接中
          <audio ref="music" loop>
    <source
        src="https://m10.music.126.net/20200407145528/70cb80be7bf7ab9c40cb3cb37dbdd6d7/ymusic/065b/520b/0409/111c898908eab454414c3c326fe77c36.mp3"
        type="audio/mpeg"
    />
</audio>

          1. 补充
          • 标签定义声音，比如音乐或其他音频流。 元素支持的3种文件格式：MP3、Wav、Ogg。
          • loop属性是循环播放

          # 解决方法二

          1. 首先 找到下载的歌曲用网页版打开复制链接中的歌曲ID 如：空山新雨后（翻自 音阙诗听）https://music.163.com/#/song?id=1397565121
          2. 然后将ID替换到链接 http://music.163.com/song/media/outer/url?id= .mp3 中，如：http://music.163.com/song/media/outer/url?id=1397565121.mp3
          3. 复制这个链接就可以直接通过网页打开纯mp3地址了，添加到下载工具中也可下载
