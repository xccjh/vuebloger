angular知识点总结

# 前言

记得在我第一次看vue文档的时候，感觉它像是在用一些朴素的语言尝试把人们的思维引导到一个新的更接地气的维度。在react的hook出来那会一次吃饭同事和我说react出来了一个新的概念hook据说很厉害。然后我跑去过了一遍react的文档发现团队维护的react最新理念貌似更新颖而先进，里面讨论的诸多概念和草案，至少在技术理念上让我感觉未来react很可能会是比vue更好的选择。然后我在恒企开始用angular进行了比较完整的项目实训。

最近又看了一遍vue3的文档，vue3的组合式api结合框架自身用新的概念借鉴了hook的形式，把应用程序分离成多个业务逻辑的聚合点，暂且抛开其他，大佬的思维让我重新审视hook的同时又一次有了新的期待。

相较而言，angular的进阶门槛可能会高一些，比如你得研究流式编程，掌握ts,学习ngrx，习惯angular带槽点的编程形式，许多官文尚未汉化，社区资料略显不足等等。有一点感觉还是挺有意思的，angular团队始终坚持视图逻辑分离的源生理念而不是jsx，他们认为jsx会把应用程序复杂化。事实上，大多数用过angular的同事都会觉得angular常常更让人匪夷所思。这大概归功于angular一站式的因素，庞大的体系做了大量的抽象和接口，概念，好像一个人要去菜市场买菜可以直接出门，而却有意识地在镜子面前梳理几下让自己在社会中更得体。不得不说，如果你深入了解angular，它会给你更多的震撼以及提升。下面我会用尽量言简意赅通俗易懂的描述分享一些我意识所触及到的对angular的见解，期望能达到略看一睹全貌，细看收获颇多的效果。接下来，让我们开始吧。

# angular简介

2009年谷歌发布Angular1，2016年9月Angular2正式版发布，Angular3由于在开发过程中出现问题（Router模块）被放弃，直接奔向4，2017年3月Angular4正式版发布，之后还有5、6、7，改动都不是很大，2019年5月28日，Angular8正式发布。截至2021年4月1日，angular最新版本angular11,ngrx11。其实这些版本中只有1.x是最特别的，因为1.x与2以及之后的版本完全不一样。

angular-cdk详解

# angular-cdk初识

cdk(Component Dev Kit组件开发包)是Angular官方在开发基于Material Design的组件库时抽象出来单独的一个开发包，里面封装了一些开发组件时的公共逻辑并且跟Material Design 设计无关，可以用来封装自己的组件库或者直接在业务开发中使用。cdk库里面的功能模块有以下几个部分：

cdk库模块 解释 cdk库模块 解释 cdk库模块 解释
Coercion 常用类型转换工具 Accessibility 放置一些方便与使用者互动的功能 Dragand Drop 组件托拽工具类
Keycodes 常用键码 Stepper 类似工作流里面经常用到的分步填写表单 Layout 响应式布局工具
Observers 监听内容变化 Platform 获取平台信息，以及平台支持的功能 Scrolling 处理滚动
Overlay 处理页面的弹出层 Portal 动态内容呈现 Table 表格
Bidirectionality 布局方向 Text field 处理文字的输入 Tree 树

angular-cli命令详解

# cli命令

# ng new/n <name> [options]

用于创建一个新的Angular项目。

options 说明
collection=collection 别名：-c。用于生成初始应用程序的外部库，比如ng-zorro-antd
inlineStyle=true|false默认值：false 别名：-s。如果为true，则在组件TS文件中包含内联样式。默认情况下，在组件TS文件中创建并引用外部样式文件
inlineTemplate=true|false默认值：false 别名：-t。如果为true，则在组件TS文件中包含模板内联。默认情况下，在组件TS文件中创建并引用外部模板文件
prefix=prefix默认值：app 别名：-p。要应用于初始项目的生成选择器的前缀
routing=true|false默认值：false 如果为true，则为初始项目生成路由模块
skipGit=true|false默认值：false 别名：-g。如果为true，则不初始化git存储库
skipInstall=true|false默认值：false。 如果为true，则不安装依赖包
skipTests=true|false默认值：false 别名：-S。如果为true，则不会为新项目生成“spec.ts”测试文件
style=css|scss|sass|less|styl默认值：css。 用于样式文件的文件扩展名或预处理器
viewEncapsulation=Emulated|Native|None|ShadowDom默认值：None 在初始项目中使用的视图封装策略
